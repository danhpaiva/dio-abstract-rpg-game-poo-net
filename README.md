<h2 id=objetivo>📜 Objetivo</h2>

Trabalhar utilizando conceitos de POO:

- Herança
- Encapsulamento
- Polimorfismo
- Abstração

<h2 id=instalacao>☑️ Instalação</h2>

1. Clone o repositório
2. Abra o projeto com o Visual Studio Code
3. Rode o projeto com:

~~~~
dotnet run
~~~~
